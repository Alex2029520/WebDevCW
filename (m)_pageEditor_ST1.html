Splash Page

Technical Description

The Splash screen uses HTML and CSS to create an interactive screenwith a striking background witha button that the user can use to take them to this page (page editor). The page then takese the user to the home page after a period of 4 seconds. The button and title are both centred in the page by using: position: relative. the font and font colour are set to white so they can be clearly viewed by the user. With the use of background-size: cover; and background-position: center the background is made to cover the entirety of the page in order to look neat and presentable

Accessibility

This page is user friendly allowing the user to clearly view the title and theme of the page with a hover feature over the page editor button to give the user the implication that the button is fully functional.

Volunteer Page

Technical Description

The volunteer page background is set to blue using background-colour:rgb(21, 38, 66); to match the theme of the rest of the webpage with the program-grid section being used to organize the images in a 2x2 grid formation for a more appealing aesthetic. I implemented .program-card .image-container:hover .overlay to adjust the transparency of the colour that appears over each image when the user hovers over them and each image displayed is set to the same height and width to give it a more appealing look.

Accessibility

The page is neatly organised so that the user has no issue navigating through the page with each volunteering position being organized neatly in a grid formation with a "Learn more" text being displayed when each image is hovered over by the user. I also left a added two section at the bottom for user reviews with an overall review being displayed above so the user is able to guage the quality and success of the provided volunteering programs.

Content Page

Technical Description

A small navigation bar that uses position: sticky; that allows it to stay visibile on the screen no matter how far the user scrolls so it is always available to them whenever they want to use it. Each image is set the the same size using width-"700" and height="500" next to each image link. A go to the top button using position: fixed; is displayed at the bottom right corner of the screen which just like the navigation bar is constantly visible on the screen regardless of how far the user scrolls. It uses href="#top" to take the user back to the top of the page.

Accessibility

A navigation bar at the top of the page allows the user to quickly take themselves to different points of the page to allow them to find relevent inforation with ease. The implementation of the go to the top button which is constantly visible on the screen allows the user to take themselves back to the top of the page without having to scroll all the way to the top themselves make the page feel less tedious to go through. All images are displayed in an organized fashion with the relevent text underneath it to clearly show which topics are linked to which images, avoiding any confusion when reading through each topic. I used

to split each group of text into sections which I seperated with a thin white line and a drop down box that allows the user to filter their search for easier navigation through the page.
